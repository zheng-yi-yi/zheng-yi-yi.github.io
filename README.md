# Zheng, Yiyi's Personal Blog

Observations on life and code. Built with [VitePress](https://vitepress.dev/).

## Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run docs:dev
   ```

## Build and Preview

1. Build the project:
   ```bash
   npm run docs:build
   ```
2. Preview the build:
   ```bash
   npm run docs:preview
   ```

## Deployment

This site is automatically deployed to GitHub Pages using GitHub Actions on every push to the `main` branch.

Check out the tutorial: [How to use VitePress](docs/how-to-use-vitepress.md)
